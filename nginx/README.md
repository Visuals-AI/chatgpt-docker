## 独立网关

------

## 说明

每台靶机的独立网关，用于把用户的请求转发到对应的场景漏洞容器服务中处理。

部署说明见 [这里](../README.md) 即可。


## 关于挂载目录

```
volumes
└── nginx
    ├── cert
    ├── etc/share
    └── html
```

- `cert` 为证书目录，当且仅当 `bin/run.sh -p https` 指定 HTTPS 协议时才需要使用证书，但一般不需要。
- `etc/share` 为漏洞场景入口的接口，按需分配即可。尽量把各个场景分配到不同的菜单中，避免接口冲突
